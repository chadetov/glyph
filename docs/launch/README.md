# Launch materials

Drafts for taking Glyph public (step 12). These are drafts to execute, not
published artifacts — the actual posting, hosting, and outreach are manual.

| File | What |
|---|---|
| [`show-hn.md`](show-hn.md) | Show HN submission (title + body) |
| [`announcement.md`](announcement.md) | Launch blog post / announcement |
| [`release-notes-v0.1.md`](release-notes-v0.1.md) | Honest v0.1 release notes |
| [`onboarding-first-100.md`](onboarding-first-100.md) | Plan to recruit and onboard the first 100 users as co-designers |

Every claim in these drafts traces to the verified evidence in
[`../../benchmarks/FINDINGS.md`](../../benchmarks/FINDINGS.md). The standing rule:
no measured-speedup claim (the agent-productivity thesis is a hypothesis), the
token numbers are an approximate proxy, and only the demonstrated verifiability
cases are claimed.

Before posting: enable GitHub Pages so the playground deploys (Settings → Pages →
source "GitHub Actions"), and link the live playground URL in the posts.
